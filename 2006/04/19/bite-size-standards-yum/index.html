Tasty Bites for Standardistas

By Ian Lloyd | April 19th, 2006 | Filed in Web Standards (general)

John Oxton reveals some tasty web standards morsels for you to have a nibble on.

Skip to comment form

It may have been a few days ago that it was revealed, but the offering is as tasty and fresh today as it was a few days ago – and should remain so for a long while yet. Enough of my food analogies, then, and on with the main course (damn, sorry): John Oxton‘s other little baby (the one that doesn’t wake up at stupid o’clock demanding attention, that is) Bite Size Standards is now live. As the title suggests, it’s all about web standards but in small, easy-to-manage nibble-sized portions. Go grab yourself a snack, or place an order to go.
