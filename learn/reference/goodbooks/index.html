Good Books

Found a bookmark?

The bookmark (PDF 3.4MB) was placed by a Member of the Web Standards Project Street Team to warn you that the book may contain information that is outdated and won’t help you to learn modern web design and development practices.

The list below contains books covering a wide variety of topics that are up to date and will ensure that the skills you are learning won’t be outdated before you learn them! There are also lots of excellent resources available free of charge online – a good starting point is the Learn section of the main Web Standards Project website.

If you would like to suggest a book for the below list, or have any concerns or questions please contact us.

If you only buy one book…

Designing with Web Standards (3rd Edition) by Jeffrey Zeldman. The foundational web standards text.

Starting out (Beginners Guides)

  • Beginning CSS Web Development: From Novice to Professional
  • Build Your Own Website The Right Way Using HTML & CSS
  • Learning Web Design, 3rd Ed.

General web design and development

  • Bulletproof Web Design: Improving flexibility and protecting against worst-case scenarios with XHTML and CSS (2nd Edition)
  • Web Design in a Nutshell: a Desktop Quick Reference 3rd edition
  • HTML Utopia: Designing Without Tables Using CSS
  • Adapting to Web Standards: CSS and Ajax for Big Sites (Voices That Matter)
  • Web Standards Solutions

CSS

  • CSS Mastery: Advanced Web Standards Solutions
  • CSS: The Definitive Guide
  • The CSS Anthology: 101 Essential Tips, Tricks & Hacks
  • The Zen of CSS Design: Visual Enlightenment for the Web (Voices That Matter)
  • CSS Cookbook
  • Transcending CSS
  • Mastering CSS with Dreamweaver CS4

HTML/XHTML

  • HTML & XHTML: The Definitive Guide (6th Edition)
  • HTML Mastery: Semantics, Standards, and Styling

JavaScript

  • Bulletproof Ajax
  • Simply JavaScript
  • The Art & Science of JavaScript
  • DOM Scripting: Web Design with JavaScript and the Document Object Model
  • The JavaScript Anthology: 101 Essential Tips, Tricks & Hacks
  • JavaScript: The Definitive Guide
  • Accelerated DOM Scripting with Ajax, APIs, and Libraries
  • JavaScript & Ajax for the Web: Visual QuickStart Guide

Web Accessibility

  • Web Accessibility: Web Standards and Regulatory Compliance
  • Just Ask: Integrating Accessibility Throughout Design
