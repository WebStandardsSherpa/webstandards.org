Αν θα θέλατε να διοργανώσετε ένα Web Standards Café, τότε μπορείτε να χρησιμοποιήσετε το υλικό που θα βρείτε πιο κάτω. Αυτό το υλικό βέβαια δεν είναι παρά μόνο η αρχή. Εσείς θα βρείτε τον τρόπο που είναι πιο αποδοτικός για εσάς.

Δε βλέπετε υλικό στη γλώσσα σας; Η WaSP είναι ένας οργανισμός που αποτελείται από εθελοντές. Οι μεταφράσεις που βλέπετε εδώ έγιναν από φίλους της WaSP. Αν θέλετε να συμμετάσχετε στη μετάφραση κειμένων, μπορείτε να επικοινωνήσετε απευθείας με τους υπεύθυνους του ILG.

Τα Web standards Café’s διοργανώνονται σε όλο τον κόσμο φέρνοντας κοντά ανθρώπους που έχουν πάθος για το Διαδίκτυο και πως θα γίνει αυτό χρήσιμο σε όλους. Πάντα ήταν ένα σημαντικό μέρος για τους επαγγελματίες του Διαδικτύου να μοιραστούν, να μάθουν και να δικτυωθούν με άλλους ανθρώπους τοπικά και με τρόπο ελεύθερο. Πρόσφατα, διάφοροι άνθρωποι αναρωτήθηκαν τι περιλαμβάνει η διοργάνωση ενός Web Standards Café, οπότε σκεφτήκαμε να γράψουμε αυτό το “Web standards Café Kit” ώστε να σας δώσουμε κάποιες ιδέες για να διοργανώσετε το δικό σας.

Αν πιστεύετε όχι έχετε εμπειρία να διοργανώσετε ή να συμμετάσχετε σε ένα Web Standards Café ατομικά ή σε γκρουπς, αφήστε ένα σχόλιο και πείτε μας τη γνώμη σας.

Ποιος

Τα Web Standards Café’s και οι σχετικές συναντήσεις διοργανώνονται από web designers και developers κι απευθύνονται σε web designers και developers που ενδιαφέρονται για τα Πρότυπα Διαδικτύου (web standards) such όπως η HTML, XHTML, XML, CSS, XSLT κλπ. και τις καλύτερες πρακτικές που τα συνοδεύουν όπως η προσβασιμότητα στο web και η ευχρηστία, η παγκόσμια φύση του Διαδικτύου και το web των ασύρματων συσκευών. Οι συναντήσεις είναι ελεύθερες και ανεξάρτητες από οργανισμούς όπως το World Wide Web Consortium (W3C) ή η WaSP.

Τι

Τα γκρουπς οργανώνουν συναντήσεις περιοδικά σε μπαρ ή καφέ ώστε να συζητήσουν συγκεκριμένα θέματα, ν’ ακούσουν σύντομες παρουσιάσεις, να συνομιλήσουν μεταξύ τους και να πιουν μερικές μπύρες. Οι συνατήσεις γενικά περιλαμβάνουν σύντομες παρουσιάσεις από εθελοντές, τις οποίες ακολουθούν ερωτήσεις κι απαντήσεις. Όσο μια κουβέντα έχει ενδιαφέρον πρακτικά δεν έχει χρονικό περιορισμό από τη στιγμή που είναι γύρω από τα standards. Κάποια από τα θέματα που έχουν παρουσιαστεί κατά καιρούς είναι: η HTML 5, η βελτιστοποίηση για τις μηχανές αναζήτησης (SEO), θέματα προσβασιμότητας, Flash κ.α.

Που

Γενικά οι συναντήσεις γίνονται σε μπαρ και σε καφέ. Αν σκέπτεστε να υποστηρίζετε παρουσιάσεις, τότε το μέρος θα πρέπει να μπορεί να υποδεχτεί ένα προτζέκτορα και την οθόνη του (τα οποία θα πρέπει να προμηθευτείτε εσείς, αν το κατάστημα δεν έχει) και πιθανόν ένα δωμάτιο στο κατάστημα μόνο για το σκοπό αυτό. Αν δεν υπάρχει ανάγκη για παρουσιάσεις, τότε μια ήσυχη γωνιά θα είναι αρκετή. Καλό είναι να ελέγξετε αν ο χώρος μπορεί να υποστηρίξει ένα τέτοιο γεγονός, ακόμα και στην περίπτωση που μια συνάντηση μετατραπεί σε ένα μεγάλο πάρτι. Κάποιες από τις συναντήσεις πιθανόν να είναι τόσο μεγάλες που να χρειάζεται μια αίθουσα πανεπιστημίου για τους χωρέσει όλους.

Πότε

Οι συναντήσεις γίνονται όσο συχνά χρειάζεται. Κάποια γκρουπς κάνουν μία συνάντηση το μήνα και κάποιοι άλλοι χωρίς πρόγραμμα. Στην ουσία όλο αυτό εξαρτάται από το ενδιαφέρον και τη διάθεση των ανθρώπων στον τόπο όπου διοργανώνεται ένα Web Standards Café.

Χορηγίες

Παρότι τα γκρουπς και οι συναντήσεις δεν είναι συνδεδεμένες με οργανισμούς, είναι πιθανό να υπάρχουν σε αυτές χορηγοί. Πολλές φορές οι χορηγίες των συναντήσεων είναι με τη μορφή φαγητού και ποτών, με τη μορφή παροχής ενός χώρου ή με την παροχή εξοπλισμού για αναπαραγωγή ήχου και video. Ιδανικά, οι συναντήσεις έχουν μια σειρά από εναλλασσόμενους χορηγούς. Αυτό είναι σημαντικό ώστε οι συναντήσεις να είναι επικεντρωμένες στο περιεχόμενο για το οποίο διοργανώθηκαν και να κρατάνε την ανεξάρτητη μορφή τους.

Προβολή

Εσείς ξέρετε τον τρόπο για την καλύτερη προβολή ενός Web Standards Café στη χώρα σας. Φυσικά, μπορείτε να δείτε λίγο πιο κάτω όπου γίνεται αναφορά σε συναντήσεις του παρελθόντος σε άλλα μέρη του κόσμου και να αποκτήσετε ιδέες σχετικά με το θέμα. Δημοφιλείς τρόποι είναι η δημιουργία ενός website για το σκοπό αυτό ή μέσω μιας απλής σελίδας στο web, οι mailing lists (π.χ. Yahoo! Groups, Google Groups ή κάτι αντίστοιχο σε κάθε χώρα), δημοσιοποίηση των συναντήσεων στο Facebook, στο Twitter και σε blogs.

Θα υπάρξει ένα Media Pack διαθέσιμο για να το κατεβάσετε στον υπολογιστή σας από το ILG BaseCamp website, το οποίο θα περιέχει αρχεία που θα σας βοηθήσουν να λύσετε όσο το δυνατό πιο εύκολα κάποια από τα συνηθισμένα θέματα που αντιμετωπίζει ένα Web Standards Café.

Τέτοια αρχεία είναι:

  • Eνα δείγμα παρουσίασης του Web Standards Café που μπορείτε να στείλετε στα τοπικά μέσα ενημέρωσης
  • Eνα δείγμα αναζήτησης χορηγίας (με έμφαση προς την αναζήτηση ενός καλού χώρου)
  • Eνα δείγμα μιας παρουσίασης που μπορεί να χρησιμοποιηθεί κατά τη διάρκεια της συνάντησης
  • Μια συλλογή από λογότυπα (του ILG και της Wasp)

Τα αρχεία δε σχεδιάστηκαν έτσι ώστε να καλύψουν κάθε πτυχή του θέματος. Αν δημιουργήσετε κάτι νέο που πιστεύετε ότι θα ήταν χρήσιμο, παρακαλούμε να το ανεβάσετε στο website.

Μετά τη συνάντηση

Προσπαθώντας να κρατήσετε το πνεύμα της συνάντησης ζωντανό, θα ήταν πολύ καλά αν μοιραζόσασταν όσα περισσότερα πράγματα μπορείτε με την ευρύτερη κοινότητα. Αυτό μπορεί να γίνει μέσω των blog σας και το ανέβασμα εικόνων, παρουσιάσεων, ηχητικών αρχείων και αρχείων video. Στην περίπτωση που κάνετε κάποιο podcast ή παρουσιάσετε κάτι σε video, θα ήταν καλό να συνοδεύεται αυτό και με την απομαγνητοφώνησή του.

Όταν είναι εφικτό, ζητήστε από τους ανθρώπους που ήταν στις συναντήσεις να ανεβάσουν σχετικό υλικό σε websites όπως το Flickr με συγκεκριμένα tags.

Κάνοντας το περιεχόμενο κοινό

Με στόχο να επιτρέψουμε σε περισσότερους ανθρώπους να συλλέξουν πληροφορίες σας ζητάμε να χρησιμοποιήσετε το υλικό σας σε κάποια από τις παρακάτω online υπηρεσίες.

Επίλογος

Αν διοργανώνετε μία συνάντηση ή έχετε συμμετάσχει σε ένα Web Standards Café στη χώρα σας κι έχετε κάποιες σκέψεις ή συμβουλές να μοιραστείτε μαζί μας, αφήστε μας ένα σχόλιο, αυτό θα μας χαροποιούσε πολύ. Επίσης, θα χαρούμε πολύ ν’ ακούσουμε για ένα καινούριο Web Standards Café σε μια περιοχή που μέχρι τώρα δεν υπήρχε. Αν είχατε κάποιες επιπλέον δυσκολίες, αν προέκυψαν θέματα που δεν είχαν προκύψει στο παρελθόν, μιλήστε μας. Θέλουμε να σας ακούσουμε.

Ο Γιάννης Κωνσταντακόπουλος σχεδιάζει κι αναπτύσει websites και γράφει κείμενα για το Web. Στο τέλος, μιλάει για όλα αυτά, με στόχο το Web να είναι χρήσιμο, εκτός από όμορφο. Είναι μέλος του ILG, που ανήκει στο Web Standards Project.
