Lucian Slatineanu is putting together a DOM script to replace the default form widgets supplied by the browser with other, more customisable elements. He calls the script Niceforms.

It isn’t done yet and, needless to say, powerful scripts like this throw up a lot of usability and accessibility questions (and probably a few browser inconsistencies too). That’s why Lucian is soliciting feedback. Try out the demo, check out the source code and then weigh in with any thoughts you might have for improving the script.
